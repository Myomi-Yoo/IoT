## View

* **`layout_width`** : view의 너비
* **`layout_height`** : view의 높이
* **`orientation`** : 배치방향
* **`id`** : 각 위젯을 식별할 수 있는 이름
  * btn
  * txt
* **`margin`** : 주위 여백
* **`padding`** : 내부 컨텐츠와 border사이의 간격
* **`layout_weight`** : 여백을 해당 view의 사이즈로 포함
* **`layout_gravity`** : parent내부에서 view의 정렬
* **`gravity`** : view 내부에서의 정렬
